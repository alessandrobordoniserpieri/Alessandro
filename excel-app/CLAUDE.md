# Excel App

App web che replica e migliora un foglio Excel con interfaccia grafica moderna.

## Stack
- React 19 + TypeScript
- Vite per build/dev
- Tailwind CSS per lo stile
- xlsx per leggere file Excel

## Convenzioni
- Rispondi e scrivi commit message in italiano
- Componenti React in `src/components/`
- File Excel di esempio in `data/`
- Stile con Tailwind, no CSS custom se non necessario
